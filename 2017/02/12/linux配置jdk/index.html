linux下配置jdk

我用ubuntu来演示(毕竟很多人也是用ubuntu。。另外最好用root除非你会用sudo)
点我进入下载jdk的页面
我用wget莫名其妙出问题，我就下载后ftp传过去的


进到目录后解压,输入

tar -zxvf jdk
然后按tab键，tab键会自动补全文件名,文件名出现后回车等待解压完成
然后创建一个文件夹存放jdk，并且把解压后的jdk移动进去
然后在root身份下(设置好root密码后终端输入su回车后输入root密码就行) 用vim来配置环境变量(/etc/profile权限用户组默认没有写权限,是只读的但是root所有者有写权限,如果没有安装vim也可以用vi,或者命令行安装vim,ubuntu,debian,kali安装时apt-get install vim -f,centos是yum install vim) 
vim /etc/profile
进入这个文件后按下i进入vim的编辑状态，移动光标在最后一行加入 在结尾处加上 
export JAVA_HOME=/usr/java/jdk1.8.0_121
export PATH=$JAVA_HOME/bin:$PATH
export CLASSPATH=.:$JAVA_HOME/lib/dt.jar:$JAVA_HOME/lib/tools.jar



然后按esc退出vim的编辑状态,然后按住shift下按下L键右侧的冒号进入vim的命令行
输入wq回车,意思是保存退出

然后输入

sources /etc/profile


然后查看java版本就成了

java -version


然后source就是在不需要重启下让配置文件生效
教程结束
