解决错误的方法就是可以手动安装任意插件

很多人在使用sublime Text3的时候多多少少会安装一些插件增加工作和使用效率,但是Ctrl+Shift+P调出命令面板输入install 调出 Install Package 选项并回车会出现package control 报错 There are no packages available for installation


网上有人说改host文件让sublime可以翻墙下载, 但是搜索了一番发现很多人这样根本无法解决,刚才自己无意中发现了一种解决方法
我的本意是安装Emmet来快速生成html头部,其他所有插件按照我下面方法一样可以安装

打开https://packagecontrol.io/channel_v3.json
是一个json文件页面,然后ctrl+f搜索Emmet

可以看出来全是github上的,然后打开图里那段md结尾的链接(markdown语法文件)

如图所示,git 克隆或者下载后打开压缩包拖到你的插件文件夹里(在sublime里点击Browse Packages就弹出插件文件夹了,中文的话就是上面菜单的首选项的第一个浏览插件),然后重启sublime就好了
